# SkyForge

## Was ist SkyForge?
SkyForge ist das zentrale Unity-basierte Entwicklungsumfeld für Drohnen-Simulations- und Steuerungssoftware bei XFLIGHT. Es dient der Integration von KI-Modellen, Sensor-Simulation und Flight-Controller-Logik in einer einheitlichen Umgebung.

## Tech Stack
- Unity Engine (aktuelle LTS-Version)
- C# für Logik und Simulation
- Hugging Face Hub für KI-Modelle
- Git LFS für große Binärdateien
- Betaflight Configurator (optional) für FC-Integration

## Quick Start
```bash
# Repository klonen (wenn bereits vorhanden)
git clone https://github.com/rudi-xflightsystems/SkyForge.git

cd SkyForge
# Submodule (falls zukünftig) initialisieren
# git submodule update --init
```
